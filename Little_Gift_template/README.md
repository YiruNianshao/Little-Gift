## ✨ 程序特点

### 🎯 核心功能
- **时段问候**：根据时间自动切换问候语
- **眼睛追踪**：眼睛会跟随鼠标移动，偷看你在干嘛
- **互动反馈**：单击身体不同部位会有不同反应（身体、脸、脑袋、小兔子耳朵）
- **特殊日期**：特别的日子会有特别的祝福哟~
- **随机对话**：其实就是文本库的弹窗啦，双击也可以触发哟
- **拖动反馈**：移动它试试看吧~

### 🎮 操作指南
| 操作方式 | 功能说明 |
|---------|---------|
| **拖动** | 移动宠物到桌面任意位置 |
| **单击身体** | 根据不同部位互动（耳朵/头/脸/身体） |
| **双击宠物** | 显示随机消息 |
| **右键菜单** | 退出程序 |
